Welcome to BlackMagic

This is a simple Music Player with added functionality, built purely using React Redux on Node.js. It allows users to

  • Play songs from your device
  • Host songs on your local network
  • Become a DJ
  • Mini radio station
  • Synchronize the same song to multiple users

React

This application relies heavily on React as a view library and Redux as a view framework. Server is build on express and socket.io is used for webSockets

Running the application

Requires node and npm to be existing on your system,

Install webpack and babel-cli

sudo npm install webpack --global
sudo npm install babel-cli --global

Running the app

cd web_app/web

npm install

npm start
